# MathKids 游戏化改造 PRD

## 目标
将现有计算练习程序从「做题工具」变成「数学冒险游戏」，让孩子像玩跑酷/消除游戏一样上瘾。

## 目标用户
- 6~12岁小学生
- 已有账号，需要激活留存
- 新用户，需要第一印象钩子

---

## 核心游戏化改造

### 1. 闯关模式（替代现有设置页）

**关卡结构** — 共5大世界，每个世界5关：
```
🌸 甜甜加法森林（加法系）
  ├─ 关1: 水果加法（10以内）
  ├─ 关2: 两位数探险（两位数加法）
  ├─ 关3: 三位数挑战
  ├─ 关4: 四位数冲刺
  └─ 关5: BOSS关（混合加法，速度挑战）

🌟 神秘减法洞穴（减法系）
  ├─ 关1~5: 对应各难度减法

✨ 乘法水晶矿（乘法系）
  ├─ 关1: 口诀入门
  ├─ 关2: 乘5/乘9速算
  ├─ 关3: 平方数
  ├─ 关4: 乘11+头同尾补
  └─ 关5: 综合乘法BOSS

🔮 除法魔法塔（除法系）
  ├─ 关1~5: 对应难度除法

🏆 速算大师赛（综合）
  └─ 随机混合题型，挑战极限
```

**关卡奖励**：
- ⭐ 3星完美通关（100%正确率）
- ⭐⭐ 2星优秀（80%+正确率）
- ⭐ 1星及格（完成但<80%）
- ⭐ 未通过（失败）

### 2. 星星收集系统（替代现有积分）

**货币体系**：
- 💎 **钻石**：顶级货币，可购买角色皮肤/特效
- ⭐ **星星**：日常奖励，完成关卡获得
- 🔥 **连胜点数**：连续正确答题积累，用于释放「技能」

**获得方式**：
- 每答对1题：+1星星
- 连续答对3题：+1连胜点数（Combo奖励）
- 关卡完美通关：额外+5~20星星
- 每日首次完成关卡：+3钻石
- 连续7天登录：+10钻石

### 3. 角色系统

**可解锁角色**：
1. 🐰 美乐蒂（默认免费）
2. 🐻 小棕熊（20钻石）- 稳定型，+2秒答题时间
3. 🦊 智慧狐狸（40钻石）- 速度型，连续答对奖励×1.5
4. 🐉 数学龙（80钻石）- 全能型，所有奖励+20%

### 4. Combo 连击系统

**视觉反馈**：
- 3连击：屏幕边缘闪蓝光
- 5连击：星星爆炸动画 + 音效升级
- 10连击：全屏特效 + 「神算子！」文字

**数值奖励**：
- Combo ≥ 3：本题得分×1.5
- Combo ≥ 5：本题得分×2
- Combo ≥ 10：本题得分×3 + 额外3钻石

### 5. 每日任务

**每日刷新任务（3个）**：
- 「今日完成3关」
- 「获得20颗星星」
- 「达成10连击」

**奖励**：完成全部3个任务 = 5钻石 + 随机道具

### 6. 道具系统

**道具类型**：
- ⏰ **时间沙漏**：+10秒（消耗：5连胜点数）
- 💡 **提示卡**：显示答案1位数字（消耗：10连胜点数）
- ⚡ **双倍奖励**：下题得分×2（消耗：15连胜点数）
- 🔄 **跳过卡**：跳过难题（消耗：8连胜点数）

**连胜点数**：每次答对+1，答错-1，不允许低于0

### 7. 统计面板升级

- 历史最高Combo
- 已通关关卡数
- 累计获得星星/钻石
- 连续登录天数
- 每周排行榜（自己vs历史最佳）
