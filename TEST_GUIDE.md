## 🔍 详细测试步骤

### 步骤1：基础导航
1. 打开应用，创建测试用户"测试员"
2. 进入主菜单，检查所有按钮状态
3. 点击"探索城市"进入城市地图

### 步骤2：城市探索测试
1. 在城市地图中查看5个区域
2. 确认第一个区域（数字广场）已解锁
3. 点击数字广场，查看安娜欢迎提示
4. 尝试点击未解锁区域，确认无法进入
5. 查看区域完成度和安娜提示

### 步骤3：口算技巧测试
1. 从主菜单进入"自由练习"
2. 选择"口算技巧"分类
3. 依次测试每种技巧：
   - 选择技巧类型
   - 回答3-5道题目
   - 检查题目生成是否合理
   - 检查答案是否正确
   - 检查步骤说明是否清晰

### 步骤4：集成流程测试
1. 从城市地图进入"数字广场"
2. 完成该区域的推荐练习（两位数加减）
3. 获得星星奖励
4. 返回城市地图，查看进度更新
5. 积累足够星星，解锁新区域

### 步骤5：边缘情况测试
1. 快速连续点击按钮
2. 输入非数字字符
3. 超时未答题
4. 网络断开（模拟离线）
5. 浏览器刷新后数据恢复

## 🐛 已知问题与解决

### 可能遇到的问题
1. **城市地图不显示**：检查cityManager导入，确认渲染函数调用
2. **技巧题目生成错误**：检查mathGen.ts中的switch语句
3. **步骤显示不完整**：检查main.ts中的getTechniqueSteps函数
4. **样式不加载**：检查style.css路径，清除浏览器缓存

### 快速修复
- 重启开发服务器：`Ctrl+C`然后`npm run dev`
- 清除浏览器缓存：`Ctrl+Shift+Delete`
- 检查控制台错误：F12打开开发者工具
